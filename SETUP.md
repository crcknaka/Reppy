# Инструкция по настройке FitTrack

## Первоначальная настройка

### 1. Установка зависимостей
```bash
npm install
```

### 2. Настройка базы данных Supabase

#### Если у вас уже есть проект Supabase:
```bash
# Перейдите в директорию supabase
cd supabase

# Сбросьте базу данных и примените новую миграцию
supabase db reset
```

#### Если вы создаёте новый проект:
```bash
# Инициализируйте Supabase (если ещё не сделано)
supabase init

# Свяжите с вашим проектом
supabase link --project-ref ваш-project-ref

# Примените миграции
supabase db push
```

### 3. Настройте переменные окружения

Создайте файл `.env` в корне проекта:
```env
VITE_SUPABASE_URL=your_supabase_url
VITE_SUPABASE_ANON_KEY=your_supabase_anon_key
```

### 4. Запустите проект
```bash
npm run dev
```

## Использование приложения

### Первый запуск

1. **Регистрация/Вход**
   - Зарегистрируйтесь или войдите в приложение

2. **Добавьте свой вес**
   - Перейдите в раздел "Прогресс"
   - Нажмите кнопку "Добавить вес"
   - Введите ваш текущий вес
   - Это необходимо для правильного расчёта объёма упражнений с собственным весом

### Создание тренировки

1. Перейдите в раздел "Тренировки"
2. Нажмите "Новая тренировка"
3. Выберите дату
4. Добавьте упражнения и подходы

### Доступные упражнения

**С собственным весом:**
- Отжимания
- Приседания
- Пресс
- Подтягивания

**С дополнительным весом:**
- Штанга лёжа
- Гантели Плечи
- Гантели Бицепс

### Отслеживание прогресса

В разделе "Прогресс" вы можете:
- Просматривать графики своего прогресса
- Отслеживать изменение веса тела
- Видеть статистику по повторениям, весу и объёму
- Фильтровать данные по конкретным упражнениям

### Расчёт объёма

**Для упражнений с весом:**
```
Объём = Повторения × Вес снаряда
```

**Для упражнений с собственным весом:**
```
Объём = Повторения × Ваш вес тела
```

Это позволяет корректно сравнивать интенсивность разных упражнений и отслеживать общую нагрузку на тренировке.

## Обновление веса тела

Рекомендуется регулярно обновлять свой вес:
1. Перейдите в "Прогресс"
2. Нажмите "Добавить вес"
3. Введите новый вес и дату замера

Приложение будет использовать последний записанный вес для расчёта объёма упражнений с собственным весом.

## Структура базы данных

### Таблицы:
- `exercises` - список упражнений (предустановленные + пользовательские)
- `workouts` - ваши тренировки
- `workout_sets` - подходы в тренировках
- `profiles` - профили пользователей (включая текущий вес)
- `body_weight_history` - история изменения веса тела

## Поддержка

Если у вас возникли проблемы:
1. Убедитесь, что все зависимости установлены
2. Проверьте переменные окружения
3. Убедитесь, что миграции применены корректно
4. Проверьте консоль браузера на наличие ошибок
